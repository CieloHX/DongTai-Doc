# 探针参数

### debug

|   属性   | 值                                                           |
| :------: | :----------------------------------------------------------- |
| 生效方式 | 重启应用，启动应用时添加“-Ddebug=”                           |
| 参数类型 | Boolean                                                      |
|   来源   | 命令行参数                                                   |
| 可选参数 | true｜false                                                  |
|  默认值  | false                                                        |
| 参数说明 | <div style="width: 300pt">开启后检测本地临时目录中是否存在核心检测引擎存在，加载本地检测引擎并启动 |

### iast.mode

|   属性   | 值                                                           |
| :------: | :----------------------------------------------------------- |
| 生效方式 | 重启应用，启动应用时添加“-Diast.mode=”                       |
| 参数类型 | 字符串                                                       |
|   来源   | 配置文件                                                     |
| 可选参数 | hunter\|normal                                               |
|  默认值  | normal                                                       |
| 参数说明 | <div style="width: 300pt">漏洞检验模式，hunter模式漏洞多误报率高，normal模式漏洞相对少误报率低 |
